​💆‍♀️ Bárbara Masajes - Landing Page
​📖 Descripción del Proyecto
​Diseño y desarrollo de una landing page enfocada en la conversión para una terapeuta de masajes. El objetivo principal de este proyecto es establecer una presencia digital sólida que permita a los usuarios conocer los servicios ofrecidos, transmitiendo relajación y profesionalismo, mientras se facilita el contacto y la reserva de turnos de manera intuitiva.
​🎯 Objetivos de UX y Negocio
​Diseño Centrado en el Usuario: Creación de una interfaz limpia y fácil de navegar que transmita confianza y bienestar visual desde el primer segundo.
​Enfoque Mobile-First: Optimización total para dispositivos móviles, garantizando una experiencia fluida considerando que la mayoría de los usuarios buscan servicios locales desde sus teléfonos.
​Llamados a la Acción (CTA) Claros: Integración estratégica de botones de contacto para maximizar las conversiones y la captación de clientes.
